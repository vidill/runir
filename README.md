# Íslenskar Rúnir

Hér eru íslenskar rúnir sem við dóttir mín gerðum. Það er hægt að nota þær í tölvu og fyrir vefi. Mappan release hefur tff og otf skrár sem þú sækir og hleður inn í stafaskrá tölvunnar. 

Icelandic runic letters for computers and web. You can use these for free. Download the files in the release folder and install on your computer to use the font. Rúnir is the icelandic word fro Runes.
